About Project

The AI-Based Student Attendance Monitoring System automatically detects student faces through a webcam, verifies their identity, and records attendance in a database. It reduces manual work, prevents proxy attendance, and provides accurate attendance reports.

System Features

Face Recognition

Automatically detects and recognizes student faces using AI.

Automatic Attendance

Attendance is marked instantly after successful face recognition.

Attendance Reports

Generate daily, monthly, PDF and Excel attendance reports.
